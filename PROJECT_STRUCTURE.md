# Economic Policy Book Project Structure

This project provides a basic template for organizing a book on economic policy. The directories include:

- `manuscript/` – draft chapters written in Markdown.
- `figures/` – charts and diagrams used in the book.
- `data/` – datasets referenced in analysis or figures.
- `references/` – citation files or bibliography management.
- `scripts/` – analysis or build scripts.
- `notes/` – planning notes and outlines.
- `build/` – generated output files.

Each folder contains a `.gitkeep` placeholder so the directory appears in version control even when empty. Chapter files under `manuscript/` start with numbering to maintain order.
