# Economics, reinvented

## Definitions
### Scope
The book will explore economics and propose realistic policies that, if implemented, would lead to a much better world / country. It will explain things at a level similar to "everything you should know about politics but don't". And it would build things from first principles.

### Audience
General public. This means people like my mom or friends. 

### Values
Think from first principles, be realistic, consider general wellbeing. Value freedom, simplicity, practicality, equality, sustainability, honesty and justice.

### Tone
The tone should be friendly and approachable. It should be very dense (go straight to the point). And more importantly, it should give hope and make people want to take action.
